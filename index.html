Tourpler - Tour Planner

Tourpler is a open source tour planner. It provides a simple dashboard and uses Google Maps for planning and displaying upcoming trips.

Feature

  • displaying and planning tours with Google Maps
  • user managment
  • comments and ticket system

Install

Dependency

  • Java
  • Play Framework

Debian/Ubuntu

Install Java runtime environment:
sudo aptitude install jre-default

Install Play Framework:
Download it from http://www.playframework.com/download
unzip play-2.2.3.zip
export PATH="$PATH:/path/to/play-2.2.3/"

Install and run Tourpler:
cd /path/to/tourpler/
play run
